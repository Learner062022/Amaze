# Amaze  

## Overview  
Amaze is a fast-paced, maze-based shooter where players must navigate interconnected sub-mazes while battling enemies to retrieve a key and **successfully reach the exit of the main maze** to complete each level. Success depends on strategic movement, effective combat, and resource management. Mini-maps assist exploration by tracking player location, key position, and movement within sub-mazes.  

## Project Description  
Players must traverse a layered maze system, moving through sub-mazes while engaging in combat to secure a key. However, the challenge isn’t just obtaining the key—it’s **making it to the end of the main maze with it while surviving enemy encounters**.  

## Key Features  
- **Multi-maze navigation:** Progress through interconnected sub-mazes to retrieve a key.  
- **Combat mechanics:** Shoot projectiles to defeat enemies along the way.  
- **Mini-map system:** Track movement within sub-mazes and key position in the main maze.  
- **Resource management:** Maintain health and ammo pickups for survival.  
- **Level progression:** The player **must reach the exit of the main maze with the key** to complete each stage.  

## Genre & Type  
- **Genre:** Maze-based shooter with action-platformer elements.  
- **Game Type:** Single-player, emphasizing navigation and combat.  

## Gameplay Mechanics  

### Player Abilities  
Players can:  
- **Shoot projectiles** to eliminate threats.  
- **Move in four directions**—forward, backward, left, and right.  
- **Collect health and ammo pickups** for survival.  
- **Use mouse-look** for precision aiming.
