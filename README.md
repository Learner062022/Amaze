# Amaze

## Description
In this Mega Man-style 2D Action Platformer, players navigate a maze of interconnected sub-mazes, battling enemies and retrieving a randomly spawned collectible to complete the game.

## Features
- Enemies.
- Jump and shoot projectiles.
- Game over/failure state.
- Main menu.
- Level end condition.
- Collectible required for game completion.
- GUI for tracking collectible.
